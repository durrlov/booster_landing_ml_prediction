End-to-End Machine Learning Project

Predicting Falcon 9 Booster Landing Outcomes

This web app uses a machine learning model trained on SpaceX data to predict whether the Rocket Booster is going to have a successful landing or not.

Multiple classification models such as Logistic Regression, Decision Tree, KNN, Random Forest, AdaBoost, Gradient Boosting, SVM, XGBoost, and CatBoost were trained and evaluated.
